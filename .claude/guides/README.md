# Guides Directory

This directory is used by agents to store design specifications and implementation guides.

## Purpose
When the `ux-design` agent creates design specifications for features, they are stored here for reference by development agents.

## Structure
Files are named: `[feature-name]-spec.md`

Example: `user-profile-spec.md`, `daily-checkin-spec.md`

## Contents
Each guide typically contains:
- UI/UX specifications
- Animation timings
- Component structure
- Platform-specific guidelines
- Accessibility requirements
